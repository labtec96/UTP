UDP-9
W dostarczonym pliku mamy podane wszyskie kraje świata wraz z byłą lub przewidywaną liczbą ludności w latach 1950-2100
Zastosowanić strumienie (pakiet java.stream)
Należy dostarczyć podane metody:
void konkretnePanstwoRok (Stream stream, String panstwo, String rok) Wyswietlająca informacje o podanym panstwie w podanym roku
void najwiekszaLiczbaLudnosc (Stream stream) Wyswietlająca nazwę panstwa rok i liczbę ludności której ludność będzie największa z całego przedziału
List String panstwaNaLitere (Stream stream, String poczatek) Zwraca listę nazw państw które zaczynają się podaną frazę. Zwracana lista powinna być posortowana w odwrotnej kolejności
List Panstwo panstwaNaLitereObiekt (Stream stream, String poczatek, String rok) Zwraca listę obiektów Panstwo panśtw które zaczynają się podaną frazą. Lista powinna być posortowana względem ludności
double liczbaLudnosciWroku (Stream stream, String rok) Zwraca liczbę ludności na świecie w podanym roku
